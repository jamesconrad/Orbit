# Orbit
A simple puzzle game about orbits

![Alt text](/Screenshots/Screenshot1.png?raw=true "Optional Title")

## Inspiration and Music Source:
Project was completed in a day, as an attempt to re-create a game from HIGHKEY Games. Their version can be found here: https://play.google.com/store/apps/details?id=com.ChetanSurpur.Orbit&hl=en

![Alt text](/Screenshots/Screenshot2.png?raw=true "Optional Title")

## Controls:
Click and drag to create an Orbiter.
Click the restart button in the top left if you are stuck.
Press Escape to quit.
